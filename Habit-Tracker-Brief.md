# Habit Tracker Brief

Use this as the input for the `brainstorm` skill during the live demo.

```text
Idea: Create a simple personal habit tracker.

I want a small app where a user can define daily habits,
mark them as done, and see their current streak.

It should be easy to use, work without mail, database,
external APIs, or other services, and help the user stay
motivated with simple progress feedback.

Focus on the main user journey: create a habit, complete it
today, see progress, and recover if a day was missed.
```
